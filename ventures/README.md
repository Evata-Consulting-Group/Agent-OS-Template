# ventures/

Early-stage, exploratory, or pre-revenue projects. This is the low-stakes sandbox
where agent-led execution is fine and nothing has hardened yet.

## Promotion rule (venture → top-level domain)

Move a venture out to its own **top-level folder** at the repo root when it earns
ANY of:
- real revenue,
- a production-tier workflow, or
- a dedicated owning agent.

Document the promotion in the move's commit message. Until then, keep iterating here
cheaply.

Each venture is just a folder: add a `CLAUDE.md` (copy `../CLAUDE_TEMPLATE.md`) once
it's substantial enough to need its own brief.
